# M-1 — Durchstich

**Fertig, wenn:** eine echte Seite vollständig über die Engine läuft — Markdown mit
`:::widget`-Blöcken, ein Live-Widget mit echter Datenquelle, Aussehen aus der Design-Bibliothek.

Erst danach wird verallgemeinert. Ein Framework, das aus null Anwendungsfällen abgeleitet wurde,
löst die Probleme, die man sich vorgestellt hat — nicht die, die auftreten.
